## 📂 Portfolio Projects

### 1 — Crude Oil Prices During Persian Gulf Conflicts
**Tools:** Excel, SQL (SQLite)

Analysed 40 years of WTI and Brent crude oil daily prices 
across four major Persian Gulf conflict periods. Used SQL 
to compare average prices, identify peak spike dates, 
measure monthly volatility and calculate post-conflict 
price recovery rates using CTEs and window functions.

**SQL concepts:** GROUP BY, correlated subqueries, 
CASE WHEN, STRFTIME, CTEs

🔗 [View Project](https://github.com/jaytare/crude-oil-gulf-conflicts)

### 2 — Persian Gulf Conflicts — Financial Market Impact
**Tools:** Excel, Tableau Public

Interactive dashboard analysing how Persian Gulf conflict 
events impacted Brent crude oil, gold and S&P 500 prices 
between 2019 and 2026. Key finding: gold gained 275% 
outperforming all assets during the Iran-Israel conflict 
period while Brent crude spiked 341% at peak war intensity.

**Tableau features:** data blending across 3 sources, 
dual axis charts, calculated fields, % change analysis, 
event severity colour gradients

🔗 [View Dashboard](https://public.tableau.com/app/profile/jay.tare/viz/ThisdashboardanalyseshowPersianGulfconflicteventsimpactedBrentcrudeoilpricesmarketfearVIXandgeopoliticalriskGPRbetween20192026_DatasourcesKaggleoilgeopoliticsdataset_/PersianGulfConflictsMarketImpact)

🔗 [View Project](https://github.com/jaytare/persian-gulf-market-impact)

### 3 — International Reactions to the Iran–Israel War 2025–2026
**Tools:** Excel, SQL (SQLite)

SQL analysis of how 210 countries and multilateral 
organisations responded to the Iran-Israel war. Key 
finding: 68 countries stayed silent — more than condemned 
Iran — revealing a deeply divided international community. 
22 of 27 EU states condemned Iran while 58.82% of 
multilateral organisations called for de-escalation.

**SQL concepts:** GROUP BY, CASE WHEN, CTEs, 
window functions OVER PARTITION BY, PRAGMA schema inspection

🔗 [View Project](https://github.com/jaytare/International-Reactions-to-the-Iran-Israel-War-2025-2026)
